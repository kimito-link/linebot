# 製品概要

## 何ができるか

「君斗りんくの送信サジェスト」は、Windows専用の常駐ツール。¥980の買い切りで、月額課金は一切ない。

マウス操作だけで4つのことができる:

1. **なぞってコピー**: 対応アプリ・ブラウザでテキストをドラッグして離すと、自動的にクリップボードへコピーされる
2. **サイドボタンで全画面スクリーンショット**: マウス側面の「戻る」ボタンを押すと、画面全体をそのまま撮影できる（Win+Shift+Sを覚える必要がない）
3. **右クリック長押し(0.35秒)で送信**: サイトに合った送信キー（Enter等）を自動で送る。0.35秒未満の短押しは、いつも通りの右クリックメニューが開く
4. **ミドルクリックでGit Bash呼び出し**: 開いていなければ起動、開いていれば前面に表示

## 誰向けか

AIチャット（ChatGPT、Claude等）やチャットツール（Chatwork等）とのやり取りが多い人。「スクショを撮って→貼り付けて→送信する」という一連の操作を、マウスだけで完結させたい人向け。

## 対応環境

Windows 10 / 11。1台のPCで利用可能。Mac・Linuxは非対応（AutoHotkeyという実装上の制約のため）。
